# Theme System Guide for Merchants

## Overview

The Ordify theme system allows you to completely customize the appearance of your e-commerce store. With our powerful theme customizer, you can match your store's design to your brand identity without any coding knowledge.

## Getting Started

### Accessing the Theme Customizer

1. Navigate to your dashboard
2. Click on "Themes" in the sidebar
3. You'll see the theme customizer interface with live preview

### Available Themes

## Theme Customization

### 1. Selecting a Theme

In the Theme Selector tab:
- Browse available themes with previews
- Click on a theme card to select it
- The change applies immediately to the preview

### 2. Customizing Colors

In the Colors tab, you can customize:

**Brand Colors:**
- **Primary Color**: Main brand color for buttons and key elements
- **Secondary Color**: Supporting color for accents
- **Accent Color**: Highlight color for badges and notifications

**Base Colors:**
- **Background**: Main background color
- **Text Color**: Default text color
- **Muted Background**: Background for cards and sections
- **Muted Text**: Secondary text color
- **Border Color**: Default border color

**Tips:**
- Use your brand's color palette
- Ensure good contrast for accessibility
- Test colors in both light and dark contexts

### 3. Typography Settings

In the Typography tab, customize:

**Font Families:**
- **Body Font**: Used for paragraphs and general text
- **Heading Font**: Used for titles and headings

**Font Sizes:**
- Adjust the scale from Extra Small to 3X Large
- Changes apply proportionally across your store

**Font Weights:**
- Control text thickness from Light to Bold
- Affects readability and emphasis

### 4. Layout Configuration

In the Layout tab, adjust:

**Container Settings:**
- **Container Width**: Maximum content width (768px - 1920px)
- **Sidebar Width**: Navigation sidebar width
- **Header Height**: Main header height

**Border Radius:**
- Control corner roundness from sharp to fully rounded
- Affects buttons, cards, and images

**Spacing:**
- Adjust the spacing scale for margins and padding
- Creates visual hierarchy and breathing room

## Live Preview

### Preview Features

- **Real-time Updates**: See changes instantly
- **Device Preview**: Switch between desktop, tablet, and mobile views
- **Interactive Elements**: Test hover states and interactions
- **Sample Content**: View with realistic product data

### Testing Your Theme

1. Switch between Products and Categories tabs
2. Test different view modes (grid/list)
3. Check mobile responsiveness
4. Verify color contrast and readability

## Saving and Publishing

### Save Your Changes

1. Click "Save Changes" to store your customizations
2. Changes are saved to your account
3. Your theme persists across sessions

### Preview Before Publishing

- Use "Preview Theme" to see in a new window
- Share preview link with team members
- Test on different devices

### Going Live

- Click "View Live Store" to see your active store
- Changes apply immediately to all visitors
- Previous settings are backed up automatically

## Best Practices

### 1. Brand Consistency
- Use your official brand colors
- Match fonts to your brand guidelines
- Keep styling consistent across all pages

### 2. User Experience
- Ensure text is readable (good contrast)
- Keep important actions prominent
- Test on multiple devices

### 3. Performance
- Avoid extremely large spacing values
- Use web-safe fonts for faster loading
- Keep customizations reasonable

### 4. Accessibility
- Maintain WCAG color contrast standards
- Use readable font sizes (minimum 14px for body text)
- Test with accessibility tools

## Troubleshooting

### Common Issues

**Theme not loading?**
- Check your internet connection
- Try refreshing the page
- Clear browser cache

**Colors not applying?**
- Ensure valid hex color codes
- Check for conflicting settings
- Try resetting to defaults first

**Preview not updating?**
- Wait a moment for changes to process
- Try switching tabs and back
- Refresh the preview panel

## Theme Variables Reference

### CSS Variables Available

All theme settings are available as CSS variables:

```css
/* Colors */
--theme-primary
--theme-secondary
--theme-accent
--theme-background
--theme-foreground
--theme-muted
--theme-muted-foreground
--theme-border

/* Typography */
--theme-font-family
--theme-heading-font-family
--theme-font-size-[xs|sm|base|lg|xl|2xl|3xl]
--theme-font-weight-[light|normal|medium|semibold|bold]

/* Layout */
--theme-container-width
--theme-sidebar-width
--theme-header-height
--theme-border-radius-[none|sm|base|md|lg|full]
--theme-spacing-[xs|sm|base|lg|xl]
```
